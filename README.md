## 这里是一份简介
**阿卡狄亚之纱** 是一款基于 Fabric 的魔法冒险休闲生存整合包，添加了刚好够用的优化模组、足够的玩法模组与部分生电模组，保证性能体验的同时使得游戏足够有趣
整合包的目标是提供一个幻想、魔法、休闲并存的冒险世界~

部分玩法模组:

- 农夫乐事 [重织] 及大量拓展包
- 植物魔法
- 神化
- 沉浸式飞机
- 更好的末地
- Macaw 系的部分家具模组
- YUNG 系的部分地形、建筑模组
- IAFCE 冰火传说社区版
- 旅行背包
- Iron Chest
- 超多生物群系
- 野性自然
- 一些个人移植的 mod
....
还有很多！

总之，
Bon voyage！
